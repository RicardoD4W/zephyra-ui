# z-icon

An icon component that allows any `material icon` powered by [google fonts](https://fonts.google.com/icons) to be used by icon name only through the `icon` property

## Properties

| Prop Name   | Type   | Default Value  | Other Values                    |
|-------------|--------|----------------|---------------------------------|
| `icon` | `string` | `info` | `success, warning, error` |

## Events
`none`


--- 

## Examples of use



    ```html
<z-icon></z-icon>
<z-icon icon="success"></z-icon>
<z-icon icon="warning"></z-icon>
<z-icon icon="error"></z-icon>

```

<WcWrapper html='<z-icon></z-icon>

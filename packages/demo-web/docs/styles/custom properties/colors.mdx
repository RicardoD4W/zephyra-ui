# Colors

These CSS variables are available at **any point in your app** from the moment you use any `zephira-ui` component, keep in mind that **you can modify them as you want** and that with the themes (light and dark) their values ​​will change, this can also be changed by CSS taking into account that we use the `[data-theme="light"]` or `[data-theme="dark"]` attribute of the `html` tag.

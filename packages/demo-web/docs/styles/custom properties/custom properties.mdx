<Translate> The CSS variables that we will see below are available at any point in your app from the moment you use any zephira-ui component, keep in mind that you can modify them as you want and that with the themes (light and dark) their values ​​will change, this can also be changed by CSS taking into account that we use the [data-theme="light"] or [data-theme="dark"] attribute of the html tag.</Translate>

<br/>

## <Translate>Example to modified a component</Translate> 

<Translate>This is an example of how you can modified default css custom properties inspect in devtools if you want to see how it works and how the css variable was overwritten only at this point</Translate>

<br/>

```html
<style>
    z-button {
        --z-color-zephyra: rebeccapurple;
    }

    z-button.red{
        --z-color-primary: palevioletred;
    }

</style>

<z-button></z-button>
<z-button class="red"></z-button>
```

## <Translate>Example to modified a theme</Translate> 

<Translate>If you want to use the zephyra-ui custom properties but you want to change it in all the document to make your own theme, provide an id to the main container of your application like in the example</Translate>

<br/>

```html
<main id="my-theme">
    <style>
        #my-theme {
            --z-color-text: yellow;
        }
    </style>

    <z-button></z-button>
    <z-alert></z-alert>
</main>
```

# z-toggle-theme

A SVG component that can be used for toggling between light and dark themes, works by the html attribute 'data-theme'

## Properties

`none`

## Events


| Event Name   | Description                             | Payload                           |
|--------------|-----------------------------------------|-----------------------------------|
| `z-theme:changed` | Fires when clicking the SVG | `detail : { dataTheme: 'light' or 'dark' }` |


--- 

## Examples of use


    ```html
    <z-toggle-theme></z-toggle-theme>
    ``` 
    
    <WcWrapper html='<z-toggle-theme></z-toggle-theme>'/>

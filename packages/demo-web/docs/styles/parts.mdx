<Translate>The CSS pseudo-element ::part represents any element within a shadow tree that has a matching part attribute.</Translate>

<Translate>All components built with Zephyra-UI have their respective assigned parts, which can be consulted in the documentation of the component or by inspecting the shadow-dom of the element itself.</Translate>

<br/>

## <Translate> Example to modified a component </Translate>

<Translate>This is an example of how you can modify the default styles with CSS parts from outside the components.</Translate>

```html
<style>
    z-button::part(button) {
        background-color: orange;
        color: black;
    }

    z-button::part(button):hover{
        background-color: yellow;
    }
</style>

<z-button></z-button>
<z-button></z-button>
```
